Proyecto Final Integrador

El proyecto final integrador es un sistema de gesti√≥n de un inventario de productos
que permite a los usuarios crear, actualizar, eliminar y consultar los productos de un inventario.
Para acceder a cada una de las funcionalidades del sistema, el usuario debe ingresar el numero
de la funcionalidad que desee realizar:
=========Menu===========
1- Registrar producto
2- Visualizar productos
3- Actualizar producto
4- Eliminar producto
5- Buscar producto
6- Repotar bajo stock
7- Salir

1- Registrar producto
El usuario debe ingresar el nombre del producto, la descripcion (opcional),
categoria, cantidad y precio.

2- Visualizar productos
El sistema muestra todos los productos registrados.

3- Actualizar producto
El usuario debe ingresar el id del producto a actualizar para buscar la
coicidencia en la base de datos.

4- Eliminar producto
El usuario debe ingresar el id del producto a eliminar para buscar la concidencia en 
la base de datos donde se encuentra la tabla de productos.

5- Buscar producto
El usuario debe ingresar el id del producto para buscar la concidencia en la base de datos.

6- Repotar bajo stock
El sistema muestra los productos con stock  igual o inferior a lo que el usuario especifique.

7- Salir
esta funcionalidad permite salir de la aplicacion cuando el usuario lo desee.
